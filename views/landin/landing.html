Nuestros valores

MISSION: Permitir a los estudiante que están finalizando su carrera estudiantil diversificada coadyuvar a tomar la mejor decisión para elegir una Carrera universitaria en función de un análisis de varios parámetros entre ellos el tipo de universidad y destrezas del estudiantes, permitiéndole tener un parámetro de su fortalezas para cursar satisfactoriamente y obtener el título universitario

VISION: Consolidarnos como la empresa de mayor soporte para los estudiantes que tienen problemas al momento de elegir una carrera universitaria, mediante el uso del portal web y la aplicación móvil creada por nuestra empresa, implementar una base de datos procesada con inteligencia artificial para tener una respuesta más clara.

Acerca de nuestro producto Consolidarnos como la empresa de mayor soporte para los estudiantes que tienen problemas al momento de elegir una carrera universitaria, mediante el uso del portal web y la aplicación móvil creada por nuestra empresa, implementar una base de datos procesada con inteligencia artificial para tener una respuesta más clara.
